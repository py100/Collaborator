<h1><a id="user-content-9月至12月内软件开发进度概览与安排" class="anchor" href="#9月至12月内软件开发进度概览与安排" aria-hidden="true"><span class="octicon octicon-link"></span></a>9月至12月内软件开发进度概览与安排</h1>
</blockquote>

<p>当前希望在2016年前完成APP的“问题与答题”部分的开发，这样安排是因为目前想要尽快开发出有一定功能，能有一定用户量的软件，方便推广和拉风头，并且在一定程度上给我们完成整个项目的开发增强了信心。</p>

<h2><a id="user-content-具体模块" class="anchor" href="#具体模块" aria-hidden="true"><span class="octicon octicon-link"></span></a>具体模块</h2>

<h3><a id="user-content-主要功能问题与答题线上线下" class="anchor" href="#主要功能问题与答题线上线下" aria-hidden="true"><span class="octicon octicon-link"></span></a>主要功能——问题与答题（线上线下）</h3>

<p><strong>发布问题</strong></p>

<pre><code>1. 题目
2. 详述
3. 所在地点\时间
4. 问题悬赏
</code></pre>

<p><strong>给予解答</strong></p>

<pre><code>1. 线上预约线下解答（主要）
2. 线上解答（多用户讨论形式，解答可点赞同与否认）
</code></pre>

<p><strong>答题悬赏</strong></p>

<pre><code>1. 问者自愿出钱（虚拟金币）
2. 问者评价
3. 相关标签指数增加
</code></pre>

<p><strong>问题圈</strong></p>

<pre><code>1. 分类显示问题（按用户喜好）
2. 问题可以点“同问”（表示关注该问题）
3. 显示热门问题（“同问”数量，评论数量综合）
4. 显示最新问题
5. 问题圈内问题界面用块状显示
    1. 标题
    2. 提问者
    3. 悬赏
    4. “同问”数
    5. 时间、地点
    6. 问题分类
    7. 问题状态（解决、未解决）
6. 单个问题显示界面
    1. 标题
    2. 提问者
    3. 内容
    4. 悬赏
    5. “同问”数
    6. 时间、地点
    7. 问题分类
    8. 问题状态
    9. 线下接单者
        1. 已解决问题显示提问者对其评价
        2. 未解决的问题显示正在解决中
    10. 线上评论
    11. 图片和附件
</code></pre>

<h3><a id="user-content-支付" class="anchor" href="#支付" aria-hidden="true"><span class="octicon octicon-link"></span></a>支付</h3>

<p><strong>由于现在申请支付宝和微信支付没有以前那么方便了，采取以下措施：</strong></p>

<p>用户间交易采取虚拟货币的形式，只是一个虚拟值的转移。</p>

<p>用户充值虚拟货币的方式为转账至我们的官方账号，然后系统为用户充值。</p>

<pre><code>1. 支付宝或微信转账接口（充值和提现）
2. 用户间虚拟货币转账
</code></pre>

<h3><a id="user-content-用户相关" class="anchor" href="#用户相关" aria-hidden="true"><span class="octicon octicon-link"></span></a>用户相关</h3>

<pre><code>1. 标签（用于标识用户喜好与擅长区域）
    - 增删查（自我设定）
    - 标签指数高于一定值得到“专家认证”
    - 特殊人群通过认证直接获得“专家认证”

2. 关注
    - 添加关注
    - 取消关注
    - 我的关注中可看关注的人相关动态

3. 登录、注册

4. 关注领域设定（关注的问题类型）
</code></pre>

<h3><a id="user-content-系统设置" class="anchor" href="#系统设置" aria-hidden="true"><span class="octicon octicon-link"></span></a>系统设置</h3>

<pre><code>1. 通知方式
    - 是否接收消息推送
    - 后台接收消息
    - 震动、声音
2. 欢迎界面（用户指南形式）
3. 账户管理功能
    - 注册
    - 登录
    - 新用户登录完善个人信息（可跳过）
        - 选择大学
        - 姓名
        - 性别
        - 学院
        - 专业
        - 年级
        - 标签设定（特殊人群专家认证）
    - 修改密码
    - 手机关联
        - 绑定手机
        - 手机通讯录匹配好友
    - 隐私设置
        - 是否愿意被关注
</code></pre>

<h2><a id="user-content-时间安排" class="anchor" href="#时间安排" aria-hidden="true"><span class="octicon octicon-link"></span></a>时间安排</h2>

<p>先开发主要功能，那些手机关联之类的其实当前有没有无所谓。</p>

<h3><a id="user-content-国庆结束之前" class="anchor" href="#国庆结束之前" aria-hidden="true"><span class="octicon octicon-link"></span></a>国庆结束之前</h3>

<pre><code>1. 注册、登录基本完成（考虑全面不出错）、
2. 发布问题、问题圈
3. 用户相关的关注领域设定
4. 用户头像设定
</code></pre>

<h3><a id="user-content-11月前" class="anchor" href="#11月前" aria-hidden="true"><span class="octicon octicon-link"></span></a>11月前</h3>

<pre><code>1. 给予解答
2. 答题悬赏
</code></pre>

<h3><a id="user-content-11月内" class="anchor" href="#11月内" aria-hidden="true"><span class="octicon octicon-link"></span></a>11月内</h3>
