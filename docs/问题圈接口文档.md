<h1><a id="user-content-问题圈接口文档" class="anchor" href="#问题圈接口文档" aria-hidden="true"><span class="octicon octicon-link"></span></a>问题圈接口文档</h1>
</blockquote>

<h2><a id="user-content-问题圈概览" class="anchor" href="#问题圈概览" aria-hidden="true"><span class="octicon octicon-link"></span></a>问题圈概览</h2>

<pre><code>1. 分类显示问题（按用户喜好）
2. 问题可以点“同问”（表示关注该问题）
3. 显示热门问题（“同问”数量，评论数量综合排序）
4. 显示最新问题
5. 问题圈内问题界面用块状显示
    1. 标题
    2. 提问者
    3. 悬赏
    4. “同问”数
    5. 时间、地点
    6. 问题分类
    7. 问题状态（解决、未解决）
</code></pre>

<h2><a id="user-content-前端" class="anchor" href="#前端" aria-hidden="true"><span class="octicon octicon-link"></span></a>前端</h2>

<p><strong>最上端右边显示用户头像点击进入设置界面</strong></p>

<p><strong>最上端中间显示倒三角，点击展开选择列表，列表中有三个tag（如下），点击后分别显示三个子页面，子页面显示不同问题</strong></p>

<pre><code>1. 我的关注（显示用户喜好的问题）
2. 热门问题（“同问”数量，评论数量综合排序）
3. 新问题（未被解决，按时间排序）
</code></pre>

<p><strong>每个方块内（一个问题显示在一个方块内）</strong></p>

<pre><code>1. 标题
2. 提问者
3. 悬赏
4. “同问”数（可以点击，点击执行“同问”操作）
5. 时间、地点
6. 问题分类
7. 问题状态（解决、未解决）
</code></pre>

<h2><a id="user-content-后端" class="anchor" href="#后端" aria-hidden="true"><span class="octicon octicon-link"></span></a>后端</h2>

<p><strong>不同tag下，提供不同显示的问题对象，注意排序的问题</strong></p>

<p><strong>对“同问”操作的反应</strong></p>

<p><strong>当某方块被点击，提供单个问题界面的跳转</strong></p>
